#Instrucciones para la ejecución del proyecto

PRE REQUISITOS:
-Tener instalado IntelliJ IDEA, de preferencia en su ultima version
-Tener Instalado y Configurado Java 8 u 11
-Tener Instalado Maven
-Contar con los Plugins de Cucumber for Java y Gherkin

PASOS PARA LA EJECUCIÓN:
1- Colocar en el RunnerTest.java el tag correspondiente. (@CHALLENGE1)
2- Configurar el RunnerTest con la versión de Java de preferencia "11" y la Class "com.bdd.runner.RunnerTest"
3- Luego de tener los listos los pasos 1 y 2 puede ejecutarse el "Run" del "RunnerTest"

PASOS PARA LA VALIDACIÓN DE RESULTADOS:
- Se puede validar los resultados al finalizar la ejecución. Deberia indicarse 1 Scenario (1 passed) y 13 Steps (13 passed)
- Se puede visualizar la evidencia de la ejecución desde:
	target/site y dando anticlip en cualquiera de los archivos "html", mismo en el que se tiene que seleccionar la 
	opción "Open In/Browser" y seleccionando el navegador de su preferencia.
